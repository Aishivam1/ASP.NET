# ASP.NET MVC 5 Welcome Application

## Overview
This is a sample ASP.NET MVC 5 (.NET Framework 4.5.2) application that displays a welcome message to users.

## Features
- ASP.NET MVC 5 Framework
- .NET Framework 4.5.2
- Bootstrap 3 Responsive Design
- Clean MVC Architecture
- Welcome Message Display
- Responsive Navigation Menu
- Multiple Action Methods (Index, About, Contact)

## Project Structure
ASPNETApplication/
├── Controllers/  
│   └── HomeController.cs
├── Views/  
│   ├── Home/  
│   │   └── Index.cshtml
│   └── Shared/  
│       └── _Layout.cshtml
├── App_Start/  
│   └── RouteConfig.cs
├── Global.asax.cs
├── Web.config
├── ASPNETApplication.csproj
└── README.md

## Prerequisites
- Visual Studio 2015 or later
- .NET Framework 4.5.2 or higher
- ASP.NET MVC 5

## Getting Started

### Step 1: Create Project
1. Open Visual Studio
2. File > New > Project
3. Select ASP.NET Web Application
4. Choose MVC 5 template
5. Name it ASPNETApplication

### Step 2: Add Files
Copy files to appropriate folders

### Step 3: Run Application
1. Press F5 to run
2. Browser opens showing welcome message

## Routes
- / or /Home/Index - Home page
- /Home/About - About page
- /Home/Contact - Contact page
